Chronic Kidney Disease Prediction

Deep learning system using CNN, LSTM and Ensemble models with 98% accuracy

Project Overview

This project develops a predictive system for early detection of Chronic Kidney Disease (CKD) using deep learning techniques. The system combines Convolutional Neural Networks (CNN), Long Short-Term Memory (LSTM) networks, and an Ensemble model to achieve high accuracy in classification.

Technical Implementation

Data Preprocessing

The dataset was sourced from Kaggle containing 400 patient records with 25 clinical features:

  • Handled missing values using linear regression for numerical data
  • Applied label encoding for categorical features
  • Normalized all features to [0,1] range
  • 80-20 train-test split with stratified sampling

Model Architectures

CNN Model Architecture

LSTM Cell Structure

  1. CNN Model:
    • 3 convolutional layers with ReLU activation
    • Kernel sizes: 32, 64, 128
    • Max pooling layers for dimensionality reduction
    • Dropout rate of 0.5 to prevent overfitting
    • Final dense layer with sigmoid activation
  2. LSTM Model:
    • 50 LSTM units with dropout rate of 0.2
    • Sequence processing for temporal patterns
    • Bidirectional wrapper for enhanced learning
  3. Ensemble Model:
    • Combined CNN feature extraction with Random Forest classifier
    • 100 decision trees with max depth of 10
    • Feature importance analysis

Key Findings

  • The Ensemble model outperformed individual models by 2-5% in all metrics
  • CNN showed better performance than LSTM for this clinical dataset
  • The system reduced false negatives by 15% compared to traditional diagnostic methods
  • Most significant predictors: serum creatinine, age, and blood pressure

Conclusion

This project successfully demonstrated the effectiveness of deep learning in medical diagnosis, particularly for Chronic Kidney Disease. The Ensemble model combining CNN and Random Forest achieved 98% accuracy, making it suitable for clinical applications.
