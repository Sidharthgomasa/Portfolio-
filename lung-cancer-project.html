Lung Cancer Detection using SVM Classification

Computer Aided Diagnosis System for early detection of lung cancer from CT scans with 98% accuracy

Download Full Report View Code

Project Overview

This project develops a Computer Aided Diagnosis (CAD) system for early detection of lung cancer nodules from chest CT images using SVM classification. The system processes medical images through denoising, segmentation, feature extraction, and classification stages to identify malignant tumors with high accuracy.

System architecture of the lung cancer detection pipeline

Key Features

  • Image Preprocessing: Total variation denoising for noise reduction
  • Lung Segmentation: Optimal thresholding and morphological operations
  • Feature Extraction: Textural and shape features from nodule regions
  • Classification: SVM with RBF kernel for benign/malignant classification
  • Accuracy: Achieved 98% classification accuracy on test dataset

Technical Implementation

The system was implemented in Python using:

  • OpenCV for image processing
  • Scikit-learn for SVM implementation
  • Matplotlib and Seaborn for visualization
  • NumPy for numerical computations

Sample classification result showing detected nodule (red) with confidence score

Project Impact

This CAD system can assist radiologists in:

  • Reducing false negatives in early-stage cancer detection
  • Standardizing diagnosis across different practitioners
  • Providing second opinions on difficult cases
  • Reducing workload in high-volume screening programs

Future Enhancements

  • Integration with hospital PACS systems
  • Mobile app for remote areas
  • 3D visualization of detected nodules
  • Multi-modal integration with patient history
